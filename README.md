# 🚀 AI Data Analysis Agent

An AI-powered system that converts natural language queries into SQL, performs analysis, and generates charts with optional forecasting.

---

## 🔥 Features

- ✅ Multi-source data ingestion (Google Drive + GitHub)
- ✅ RAG (Retrieval Augmented Generation) using FAISS
- ✅ Natural language → SQL using Gemini API
- ✅ SQLite database integration
- ✅ Automatic chart generation
- ✅ Time-series forecasting using Prophet

---

## 🧠 Tech Stack

- Python
- FastAPI
- SQLite
- FAISS (Vector Search)
- SentenceTransformers (Embeddings)
- Google Gemini API (LLM)
- Prophet (Forecasting)
- HTML, CSS, JavaScript (Frontend)
